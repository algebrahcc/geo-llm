# Agent Browser

This skill provides capabilities to browse the web, search for information, and read documentation.

## Core Capabilities

### 1. Web Search

- Perform searches to find latest information, documentation, or solutions
- Combine results from multiple sources
- Validate information across different sites

### 2. Documentation Reading

- Read and extract information from official documentation
- Parse API references and guides
- Find code examples and patterns

### 3. Content Extraction

- Summarize web pages
- Extract specific data points
- Analyze web content for relevance

## Usage Guidelines

When invoking this skill, consider:

- **Search Queries**: Be specific with search terms
- **Context**: Provide context about what information you're looking for
- **Verification**: Cross-reference critical information

## Example Scenarios

- "Check the latest documentation for Supabase Auth"
- "Find how to implement drag and drop in Vue 3"
- "Search for best practices in Nuxt 4 middleware"
